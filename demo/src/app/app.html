Theme behavior

The shell flips between light and dark.
The button keeps the same logic API in both modes.
Brand 1 colors come entirely from the imported CSS pack.
The current PoC deliberately ships only the primitive represented in Figma: the button.

What proves the separation

The Angular wrapper provides framework integration only.
The imported CSS library gives the primitives their brand and surface tokens.
The same wrapper logic swaps between light and dark with CSS alone.
Host classes and per-instance CSS properties still tailor the component locally.
Removing a non-Figma primitive leaves the Angular bridge untouched for the shipped button.
