# @fuggetlenfe/angular-wrapper

Angular standalone wrapper for `@fuggetlenfe/components` – auto-generated by Stencil Angular output target.

## Exports

- **`.`** — Angular standalone directives (e.g., `FfButton`)
- **`./server`** — `renderToString` / `hydrateDocument` for SSR with Declarative Shadow DOM

## Usage

```typescript
import { FfButton } from '@fuggetlenfe/angular-wrapper';

@Component({
  standalone: true,
  imports: [FfButton],
  template: `<ff-button variant="primary">Click me</ff-button>`
})
export class AppComponent {}
```
